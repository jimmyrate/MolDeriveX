# MolDeriveX

MolDeriveX: Antibiotic Derivatization Based on Multi-Objective Molecular Generation Algorithm Expand Expert Knowledge

MolDeriveX is an AI-driven generative framework designed to navigate the complex multi-objective derivatization space of antibiotics. By coupling trust-region local search with gradient-guided global constrained generation, it elegantly circumvents local optima to discover broad-spectrum therapeutics while maintaining strict parent scaffold fidelity.
